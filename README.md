# Engelina

## .env example
```
# Database
POSTGRES_USER=...
POSTGRES_PASSWORD=...
POSTGRES_DB=database

# Logging
LOG_FILE=''
LOG_FORMAT='%(filename)s:%(lineno)d #%(levelname)-8s [%(asctime)s] - %(message)s'

# Bot
TOKEN=...

# Backend
DATABASE_URL=postgresql+asyncpg://...:...@db:5432/database

```
